# BellChecker
Hash-Based File Integrity Checker (C language, MD5)

# Notice
>>The program requires the full file path to locate and process the selected file. Please enter the complete path to continue.

# Known Limitation
>>Automatic conversion from Windows file paths (e.g., C:\Users\...) to WSL paths is currently under development and may not work as expected. When running the program in WSL, please enter the Linux-style path (e.g., /mnt/c/Users/...) instead.
